# モンスター対戦アプリ - プロジェクト仕様書

## 企画

### コンセプト
> 読み合いに全振りした、シンプルだけど深い対戦ゲーム

### ターゲット
- 20〜30代のゲーマー
- 対戦ゲーム好き（格ゲー、カードゲーム、ポケモン対戦等）
- 反射神経よりも「読み」で勝ちたい人

### 体験価値
- 相手の裏をかいた時の快感
- 「読まれた！」「読み勝った！」の駆け引き
- 友達との対戦で盛り上がる

### 差別化ポイント
- 反射神経不要（誰でも土俵に立てる）
- 1戦2分で終わる手軽さ
- 7コマンド×2のじゃんけん構造による読み合い

---

## 概要

モンスターを1対1で戦わせる対戦アプリ。
バトルシステムは「ドラゴンシーズ 最終進化形態」をベースに、オリジナル要素（スタンスシステム）を追加。

### 対応プラットフォーム

**第1フェーズ: Webアプリ**
- ブラウザで動作（PC/モバイル両対応・レスポンシブ）
- インストール不要。URLシェアでローカル対戦が容易
- 開発コストを抑えつつ、早期にユーザーの反応を確認

**第2フェーズ以降: モバイルアプリ化を検討**
- 需要に応じてPWAまたはネイティブアプリ（iOS/Android）化
- Steam等も反応次第で検討

## バトルシステム

### TCB（ツインコマンドバトル）

毎ターン、7つのコマンドから **2つを順番に選択** して同時実行する。

#### 実行順序
1. 双方の1stコマンドを同時処理
2. 双方の2ndコマンドを同時処理

#### コマンド一覧

| # | コマンド | 効果 |
|---|---------|------|
| 1 | 前進 | 相手との距離を1段階縮める |
| 2 | 後退 | 相手との距離を1段階広げる |
| 3 | 武器攻撃 | 近距離で物理ダメージを与える |
| 4 | 特殊攻撃 | 全距離で使用可能な攻撃 |
| 5 | リフレクター | 特殊攻撃を防御/反射する |
| 6 | スタンスA | スタンス切替（現在のスタンスにより表示変動） |
| 7 | スタンスB | スタンス切替（現在のスタンスにより表示変動） |

### バトル開始条件

| 項目 | 内容 |
|------|------|
| 初期距離 | **中距離** |
| 初期スタンス | 通常 |
| 制限時間 | 2分（コマンド入力時間は除外） |

---

### 距離システム

3段階: **近距離 / 中距離 / 遠距離**

- 1コマンドで1段階移動
- 双方が同方向に移動した場合は2段階移動する

#### 距離変動表

| A | B | 近→ | 中→ | 遠→ |
|---|---|-----|-----|-----|
| 前進 | 前進 | 近(変わらず) | 近(-2) | 近(-2) |
| 前進 | 後退 | 近(相殺) | 中(相殺) | 遠(相殺) |
| 後退 | 前進 | 近(相殺) | 中(相殺) | 遠(相殺) |
| 後退 | 後退 | 遠(+2) | 遠(+2) | 遠(変わらず) |

---

### コマンド優先順位（じゃんけん構造）

```
武器攻撃 > リフレクター > 特殊攻撃
```

- **武器攻撃 > 特殊攻撃**: 近距離で同時に出した場合、武器が先に発生し特殊を潰す
- **武器攻撃 > リフレクター**: 武器攻撃はリフレクターで防げない（貫通）
- **リフレクター > 特殊攻撃**: 特殊攻撃をリフレクターで無効化/反射

## パラメータ

### モンスターパラメータ（第1フェーズ: 種族ごと固定値）

| パラメータ | 役割 |
|------------|------|
| **HP** | 体力。0で敗北 |
| **腕力** | 武器攻撃のダメージに影響 |
| **特殊** | 特殊攻撃のダメージに影響 |
| **素早さ** | 回避率に影響（唯一の回避要素） |
| **丈夫さ** | 防御力。武器攻撃の被ダメージを軽減 |

### ダメージ計算式

| 攻撃種別 | 計算式 |
|----------|--------|
| 武器攻撃 | `max(腕力 × 武器倍率 - 丈夫さ, 1)` |
| 特殊攻撃 | `特殊 × 技倍率` |
| 特殊攻撃（回数超過時） | `特殊 × 技倍率 × 0.5` |
| リフレクター反射 | `相手の特殊ダメージ × 反射率` |

#### 技倍率
- **1.0**（特殊パラメータの値がそのままダメージになる）

#### 補足
- **丈夫さは武器攻撃にのみ有効**。特殊攻撃は丈夫さで軽減できない（リフレクターで防ぐ）
- 最低ダメージは **1**（0以下にはならない）
- スタンスによる倍率はダメージ計算時に適用される

### 回避

| 項目 | 内容 |
|------|------|
| 計算式 | `min(素早さ × 0.5, 25)%` |
| 上限 | **最大25%** |
| 対象 | 武器攻撃・特殊攻撃どちらも回避可能 |
| 回避時 | ダメージ0 |
| 例外 | **近距離で武器 vs 武器は回避不可（必中相打ち）** |

#### 回避率の目安

| 素早さ | 回避率 |
|--------|--------|
| 10 | 5% |
| 20 | 10% |
| 30 | 15% |
| 40 | 20% |
| 50+ | 25%（上限） |

## 装備（第1フェーズ: キャラ固定）

### 武器

| 項目 | 内容 |
|------|------|
| 名前 | キャラごとに固有 |
| 武器倍率 | ダメージ計算に使用 |

### リフレクター

| 項目 | 内容 |
|------|------|
| 名前 | キャラごとに固有 |
| 反射可能回数 | 反射成功時に1消費 |
| 反射率 | 相手の特殊攻撃ダメージに対する反射ダメージの割合（0.0〜1.0） |

#### リフレクター動作詳細

| 反射回数 | 動作 |
|---------|------|
| 1以上 | 特殊攻撃を**無効化** + **反射ダメージ**（回数を1消費） |
| 0 | 特殊攻撃を**無効化のみ**（反射ダメージなし） |

---

## 特殊攻撃

| 項目 | 内容 |
|------|------|
| 仕様 | 全キャラ共通（エフェクトは種族ごとに異なる） |
| 倍率 | 技倍率 1.0（特殊パラメータがそのままダメージ） |
| 使用回数 | キャラごとに固定値（＝全力で打てる回数） |
| 回数超過時 | 攻撃力が **半減** する |

## スタンスシステム（オリジナル要素）

### 概要

バトル中にスタンスを切り替えることで、攻撃力と防御力のバランスを変化させる。

### スタンス一覧

| スタンス | 腕力・特殊 | 丈夫さ |
|---------|-----------|--------|
| **通常** | ×1.0 | ×1.0 |
| **攻勢** | ×1.3 | ×0.7 |
| **守勢** | ×0.7 | ×1.3 |

### 仕様

| 項目 | 内容 |
|------|------|
| 初期状態 | 通常 |
| 切替方法 | どのスタンスからでも1回で任意のスタンスに切替可能 |
| 切替コスト | コマンド1枠消費（無防備になる） |
| 被弾時 | 攻撃を受けてもスタンス切替は中断されない |
| 維持 | 再度切り替えるまで永続 |

### コマンドUI表示

| 現在のスタンス | スタンスA表示 | スタンスB表示 |
|--------------|-------------|-------------|
| 通常 | →攻勢 | →守勢 |
| 攻勢 | →通常 | →守勢 |
| 守勢 | →通常 | →攻勢 |

## 勝敗条件

| 条件 | 結果 |
|------|------|
| 相手のHPを0にする | 勝利 |
| 制限時間（2分）終了時、HP残量が多い方 | 勝利 |
| 制限時間終了時、HP同値 | **ドロー** |
| ギブアップ | 敗北 |

### 補足
- コマンド入力時間は制限時間に含まない
- 第1フェーズではキャラのロスト（永久消失）なし

## ゲームモード（第1フェーズ）

### 挑戦モード

| 項目 | 内容 |
|------|------|
| ステージ数 | 8ステージ |
| 1ステージ | 1戦 |
| キャラ選択 | 解放済みキャラから1体選択 |
| 再挑戦 | 何度でも可能 |
| クリア報酬 | 敵キャラの使用解放 + 全キャラ能力UP |

#### ステージ順序

| ステージ | 対戦相手 | CPU AIレベル | 設計意図 |
|---------|---------|-------------|---------|
| 1 | ゴーレム | Lv1 | チュートリアル。鈍重で倒しやすい |
| 2 | トレント | Lv1 | HP型。長期戦の感覚を学ぶ |
| 3 | ウィスプ | Lv2 | 特殊攻撃型。リフレクターの重要性を学ぶ |
| 4 | 大亀 | Lv2 | 防御型。特殊攻撃で崩す戦略を学ぶ |
| 5 | ワイバーン | Lv3 | 素早さ型。回避への対応を学ぶ |
| 6 | ミノタウロス | Lv3 | 物理特化。近距離の脅威に対処する |
| 7 | フェニックス | Lv4 | 魔法特化。遠距離からの猛攻に耐える |
| 8 | 剣士（強化版） | Lv4 | バランス型の強敵。総合力が試される |

#### ステージ1: チュートリアル

ステージ1（vs ゴーレム）は**ターンベースの完全固定チュートリアル**。
ターン1〜5はプレイヤー・ゴーレム双方のコマンドが固定され、各コマンドの効果を確実に体験できる。

| ターン | プレイヤー（剣士） | ゴーレム | ポップアップ | 体験内容 |
|--------|-------------------|---------|-------------|---------|
| 1 | 前進・前進 | 前進・前進 | 「前進で距離を詰めよう」 | 距離の概念を学ぶ |
| 2 | 武器攻撃・武器攻撃 | 前進・前進 | 「武器攻撃で攻撃しよう」 | 近距離で武器が当たる |
| 3 | 特殊攻撃・特殊攻撃 | 後退・後退 | 「特殊攻撃は距離に関係なく当たる」 | 離れた相手にも命中 |
| 4 | リフレクター・リフレクター | 特殊攻撃・特殊攻撃 | 「リフレクターで特殊攻撃を跳ね返せ」 | 反射を体験 |
| 5 | スタンスA・武器攻撃 | 前進・前進 | 「スタンス切替で攻撃力UP」 | 攻撃的スタンスの火力を体感 |
| 6以降 | **自由** | **CPU Lv1** | 「自由に戦ってみよう！」 | 自由戦闘 |

- ターン1〜5: コマンド選択UIは表示するが、選択肢は固定（1つのみ or 自動選択）
- ターン6以降: 全コマンド解放、CPU Lv1（ランダム行動）で実戦

#### 難易度設計（CPU AIレベル）

| レベル | 行動ロジック | 該当ステージ |
|--------|-------------|-------------|
| **Lv1** | ほぼランダム行動。重み付けなし | 1-2 |
| **Lv2** | 距離に応じた基本行動（近距離→武器攻撃を選びやすい等） | 3-4 |
| **Lv3** | Lv2 + 相手のHP残量・スタンス・リフレクター残数を考慮 | 5-6 |
| **Lv4** | Lv3 + プレイヤーの行動パターンを読む高度なAI | 7-8 |
| **Lv5** | 常に最適行動を選択。ランダム要素なし | 自由対戦のみ |

#### AI Lv4 詳細（プレイヤー行動の読み）

| 項目 | 内容 |
|------|------|
| 履歴参照 | 直近3ターン |
| 分析対象 | 距離別のコマンド頻度 |
| 対応 | 頻出コマンドへのカウンター選択 |
| ランダム要素 | 20%でLv3相当の行動（完全予測防止） |

#### 種族別AI傾向

**全AIレベル共通で適用**。各種族の強みを活かした行動に重み付けを行う。
AIレベルが上がると、この傾向をより賢く・的確なタイミングで活用する。

| 種族 | 傾向 |
|------|------|
| ゴーレム | 前進＋武器攻撃重視 |
| トレント | 守勢＋距離維持 |
| ウィスプ | 後退＋特殊攻撃重視 |
| 大亀 | 守勢＋リフレクター重視 |
| ワイバーン | バランス＋スタンス切替活用 |
| ミノタウロス | 攻勢＋前進＋武器攻撃重視 |
| フェニックス | 遠距離維持＋特殊攻撃連打 |
| 剣士 | バランス型（状況対応） |

#### キャラ能力UP（ステージクリアボーナス）

- 各ステージクリアごとに **全手持ちキャラの能力が上昇**（最大7段階）
- 上昇量は **種族の特徴に準じた偏り** で上がる（腕力型は腕力が大きく伸びる等）
- 敵キャラもステージ進行に応じて能力が上昇（常にプレイヤーと同等レベル）
- 種族ごとの能力の偏りが段階的に顕著になる

##### 1段階あたりの能力UP値

| 種族 | HP | 腕力 | 特殊 | 素早さ | 丈夫さ |
|------|----|------|------|--------|--------|

| ゴーレム | +15 | +8 | +2 | +1 | +5 |
| ウィスプ | +10 | +1 | +8 | +4 | +2 |
| ワイバーン | +12 | +4 | +4 | +5 | +3 |
| 大亀 | +15 | +3 | +2 | +1 | +8 |
| トレント | +20 | +2 | +3 | +2 | +5 |
| ミノタウロス | +13 | +7 | +1 | +2 | +6 |
| フェニックス | +10 | +1 | +8 | +5 | +1 |

※ 最大7段階UP（全8ステージクリアで7回成長）

### 自由対戦（CPU）

| 項目 | 内容 |
|------|------|
| 自キャラ選択 | 解放済みキャラから選択（ランダム選択あり） |
| 相手キャラ選択 | 自分で選ぶ or ランダム |

| キャラ能力 | **最終パラメータ（全ステージクリア後の値）で固定** |
| 勝敗記録 | なし |

### 自由対戦（ローカルオンライン）

| 項目 | 内容 |
|------|------|
| 通信方式 | **同じWi-Fi内 + WebSocket**（第1フェーズ） |
| マッチング | 部屋を立てる → URLをシェア → 相手がアクセスして入室 |
| パスワード | オプション（つけたい人だけ） |
| 制約表示 | **同じWi-Fi内でのみ対戦可能であることをゲーム内に明示** |
| コマンド入力 | お互い見えない |
| キャラ能力 | **最終パラメータ（全ステージクリア後の値）で固定** |
| キャラ解放 | 挑戦モードの進捗と連動 |
| 勝敗記録 | **裏で記録**（画面表示なし。将来表示の可能性あり） |

#### 制限時間の計測（ローカルオンライン）

| 項目 | 内容 |
|------|------|
| 時間管理 | サーバー側で経過時間を計測 |
| 入力待機中 | 時計停止（双方のコマンドが揃うまで） |
| 入力タイムアウト | 30秒（超過時は前ターンと同じコマンドを自動選択） |

## 技術スタック

### フロントエンド
- **Phaser.js**（2Dゲームエンジン）
- **TypeScript**
- ホスティング: **Vercel**（無料枠）

### バックエンド（ローカル対戦用）
- **Node.js + Socket.io**
- **TypeScript**
- ホスティング: **Render**（無料枠）

### 選定理由
- Phaser.js: バトル演出（攻撃エフェクト、ダメージ表示等）の実装が容易
- Socket.io: WebSocket通信の定番。実績豊富で情報が多い
- TypeScript: 型安全性。フロント・バック共通で使用可能
- Vercel/Render: 無料枠で十分。スケール時は有料プランへ移行
